# vue商城项目

## 项目依赖
```
npm install
```

### 运行项目
```
npm run serve
```

### 项目打包
```
npm run build
```


### 项目目录说明
```
src
    assets  资源
        img  图片资源
        css  css资源

    components  组件
        common  完全公共组件
        content  当前项目内公共组件

    views 大的视图
        home  主页
        category  分类

    router  路由

    store  vuex状态管理

    network  网络请求

    common  公共js文件
    
```
